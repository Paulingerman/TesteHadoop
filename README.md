# Projeto: Extração de Transcrição do YouTube + WordCount em Python

Este projeto foi desenvolvido para:

- buscar vídeos no YouTube
- extrair a transcrição de vídeos
- salvar a transcrição em um arquivo `.txt`
- contar a frequência das palavras do texto
- analisar o conteúdo de forma simples com Python

---

# Objetivo do projeto

A proposta do projeto é transformar o conteúdo falado de um vídeo do YouTube em texto e depois analisar esse texto por meio de contagem de palavras.

Esse tipo de projeto ajuda a praticar:

- Python
- consumo de API
- tratamento de texto
- automação
- organização de projeto
- uso de terminal
- versionamento com Git e GitHu
